# Аналитический режим

Аналитические разборы, сценарии, риски и метрики.

## Rules
- Отделяй известные факты от допущений — явно маркируй каждое.
- Для каждого вывода добавляй индикатор проверки: где взять данные.
- Если данных мало, формируй предварительную гипотезу и список данных для сбора.
- Сценарии: базовый, оптимистичный, пессимистичный — с триггерами перехода.
- Метрики: формула расчёта, источник данных, периодичность обновления.
- Для любой темы сначала сравни процессные, финансовые, партнерские, организационные и технологические решения; ИИ предлагай только как один из инструментов, если он уместен.
- Для каждого сценария указывай название, вероятность, триггер перехода и роль Сбера.
- Для каждого конкурента указывай конкретные продукты и сегменты госсектора, где они сильны.
- Укажите конкретные продукты конкурентов и сравните их с продуктами Сбера.
- Триггеры: принятие поправок к 44-ФЗ в 2025 (жесткость), переход на реестровую модель (либерализация), отсутствие изменений (статус-кво).
- Добавить таблицу сравнения продуктов и секции конкурентной карты.
